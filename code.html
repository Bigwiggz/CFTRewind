The Code for Rewind


                        // Rewind Javascript

                        //Event Listener
                        let runRewind=document.getElementById("btnSubmit");
                        runRewind.addEventListener("click", StartRewind);
                        
                        //Main Function
                        function StartRewind(){
                            
                            //Get input values
                            let inputText=document.getElementById("inputText").value;
                        
                            //Validate text entry to assure there is at least 1 character
                            if(inputText.length===0){
                                errorMessage="Please put in at least one character.";
                                let errorMessageDisplay=document.getElementById("validation-summary");
                                errorMessageDisplay.innerHTML=errorMessage;
                            }
                            
                            //Call Reverse String function
                            let reveresedString=GetReverseString(inputText);
                            
                            //Place on screen
                            ResultToScreen(reveresedString);
                        }
                        
                        // function to reverse string
                        function GetReverseString(inputText){
                            let reversedString=[];
                            for(let i=inputText.length-1;i>=0 ;i--){
                                reversedString.push(inputText[i]);
                            }
                            
                            return reversedString;
                        }
                        
                        //Place result of character array to screen
                        function ResultToScreen(outputText){
                            let fullString=outputText.join("");
                            let outputField=document.getElementById("outputTextField");
                            outputField.innerHTML=fullString;
                        }

The Code is structured in one function.

Event Listener

This is the javascript event listener for the "Run Program" button to start the Code Rewind Program

StartRewind

This function gets the input text from the text box and stores it in a variable of type string. Then it runs a validation check to see if at least one character is entered in the text box. If there is no characters, then an error message is triggered. If there is no error message, then the GetReverseString method is called to reverse the letters and the ResultToScreen method is used to display the results on the screen.

GetReverseString

This method accepts a string input and parces all of the characters in the string in an array, then it reverses the order using a for loop to reverse all the characters in the string. The output is an array of reversed characters.

ResultToScreen

This method accepts outputText as an array of characters and joins them together then takes the resulting string and adds it to the HTML element with an id of "outputTextField".
